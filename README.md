# Personal Finance Plugin — Claude Code Plugin Marketplace

Tax analysis and wealth planning plugins for solopreneurs and business owners.

## Install

```bash
/plugin marketplace add benstauffer/personal-finance-marketplace
/plugin install personal-finance
```

## Plugins

### personal-finance (v0.2.0)

Client-facing tax analysis and planning for business owners:

- `/tax-review` — Upload an IRS transcript for instant situation review and optimization opportunities
- `/estimate` — Run tax estimates with actionable recommendations
- `/entity-compare` — Compare S-Corp vs Schedule C with implementation roadmaps
- `/projection` — Build 5-year projections with year-by-year action plans
